## 训练配置说明

### 推荐训练参数（CIFAR-10）
```python
TrainingConfig(
    epochs=100,           # 推荐 100-200
    batch_size=64,        # GPU: 64-128, CPU: 16-32
    learning_rate=1e-4,   # 初始学习率
    device='cuda',        # 推荐使用 GPU
    mixed_precision=True  # 启用混合精度加速
)
```

### 推荐训练参数（Tiny ImageNet）
```python
TrainingConfig(
    epochs=150,           # 推荐 150-300
    batch_size=32,        # 较大的图像需要更小的 batch
    learning_rate=5e-5,
    device='cuda',
    data_augmentation=True
)
```

## 当前限制和改进方向

### 当前限制
1. **训练脚本需要 GPU**: CPU 训练非常慢（~12 小时/epoch）
2. **FID/IS 需要预训练 InceptionV3**: 暂未包含
3. **完整训练需要大量时间**: 建议至少 100 epochs

### 下一步改进
1. ✅ 添加预训练 InceptionV3 用于 FID/IS 计算
2. ✅ 实现分布式训练支持（DDP）
3. ✅ 添加更多可视化（t-SNE、训练曲线）
4. ✅ 完善检查点管理和自动恢复

## 使用指南

### 快速开始（CPU 测试）
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行测试
python quick_test.py

# 3. 运行训练（CPU，演示用）
python train_final.py --epochs 5 --batch_size 16
```

### 完整训练（推荐 GPU）
```bash
# CIFAR-10
python train_final.py --dataset cifar10 --epochs 100 --batch_size 64 --gpu

# Tiny ImageNet
python train_final.py --dataset tiny_imagenet --epochs 150 --batch_size 32 --gpu
```

### 评估模型
```bash
# 使用训练好的模型生成图像
python evaluate_model.py --checkpoint outputs/checkpoints_CIFAR-10/checkpoint_epoch_100.pt

# 计算 FID/IS（需要 InceptionV3）
python calculate_metrics.py --real data/cifar10/test --generated outputs/generated/
```
